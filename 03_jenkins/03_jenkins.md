# Jenkinsとは？

<br/>

* Java製のCIツール
* 元々はHudsonというツールだったが、Ora<span class="weak">（お察しください）</span>
* 様々な作業が自動化できる
    * ビルド
    * テスト
    * デプロイ
    * プラグインを突っ込めば可能性は無限大！

!SLIDE

# Jenkinsのインストール

<br/>

1. OS/ディストリビューションのパッケージからのインストール
    * yum
    * apt-get
    * homebrew
2. 公式からインストーラを落としてくる
3. 公式からバイナリ（war）を落としてくる
    * winstoneでの単体起動
    * GlassFish/Tomcat/Jettyなどにデプロイ
